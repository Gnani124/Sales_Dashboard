\# 📊 Sales Dashboard | Power BI Project



\## 🚀 Overview

This project presents a comprehensive \*\*Sales Dashboard\*\* built using Power BI to analyze business performance and financial metrics.



\---



\## 📌 Key Insights

\- 💰 Total Sales, Profit, Cost, and Margin analysis

\- 📈 Year-over-Year (YoY) performance tracking

\- 📊 Sales trends across Year, Quarter, and Month

\- 🛍 Category-wise and Product-wise performance

\- 🌍 Regional sales insights

\- 👥 Top-performing segments analysis



\---



\## 🛠 Tools \& Technologies

\- Power BI

\- DAX (Data Analysis Expressions)

\- Excel (Data Source)
